Tarayıcılar arası tutarlı çalışan kod yazmak

  • Tarayıcı varsayılı stillerinin sıfırlanması (reset css)
  • Yeni css özelliklerinin uygunluk tablolarını kontrol et - Check Browser compatibility ( mdn, caniuse.com...)
  • Gerekli durumlarda vendor (tarayıcı hizmeti sağlayıcılar) prefix kullan... ( -webkit, -moz, -o, -ms )
  • preprocessor kullanarak - yeni özellikleri daha güvenli bir şekilde kullanabilirsin ( sass .scss)
  • Farklı tarayıcılarla testler yapılabilir
  • @supports css rule kullanabilirsin
