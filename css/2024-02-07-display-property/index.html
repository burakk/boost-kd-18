Css display özelliği

Etiketlerin display özellik değerleri default olarak aşağıdaki değerlerden biri olabilir.

  • display:inline
  • display:block
  • display:inline-block
  • display:none

Duruma göre daha komplex display değerleri kullanılabilir.

  • display:flex
  • display:grid

inline (satır içi) öğeler

  • inline öğeler satır okuma yönünde hizalanırlar
  • width ve height verilemez. Bu etiketler alan etiketleri olarak düşünülmemiştir.
  • default genişlikleri auto'dur. İçeriklerine göre otomatik değişir.
  • Yükseklikler satır yüksekliğine bağlıdır. (line-height)
  • Inline öğeler, kapsayıcılarından birine verilen text-align özelliğinin left, right, center, justify değerleri kullanılarak okuma yönünde hizalanabilirler
  • vertical-align özelliğinin top, middle, bottom değerleri ile satır içerinde dikeyde hizalanabilirler

block öğeler

  • block yönü dikey yöndür. Block öğeler alt alta dizilme eğilimindeler
  • default genişlikleri 100%
  • default yükseklikleri auto'dur
  • width ve height verilebilir.
  • Block öğelerden kalan pozitif boşluk - pozitive space, margin sağ ve sol değeri auto yapılarak paylaştırabilir.
